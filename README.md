## CV Backend Nodejs Express

Technologies:

- nodejs with Expres
- jwt-simple
- moment
- cors
- mongoose
- heroku

## How to start up

- Install dependencies
```bash
$ npm install
```
- Run project
```bash
$ npm run startdev
```

## How to Deploy in Her

- Deploy
```bash
$ git push heroku master
```

- View Logs
```bash
$ heroku logs --tail
```
